Веб приложение для игры в Тайного санту. 
Цель: получение практических навыков, разработка микросервисной архитектуры при помощи язык Python, реализовать взаимодействие между сервисами используя фреймворк FastAPI.
Разработал три микросервиса, настроил Nginx в качестве APIGateway и раздачи статики.
Регистрация и аунтификация с JWT токенами, валидация данных при помощи схем библиотеки Pydantica, использовал легковесную базу данных SQLite и 
взаимодействовал с ней при помощи моделей ОRМ SQLAlchemy.
Поднятие Docker контейнеров при помощи Docker-compose.
Первый сервис для пользователей: регистрация, аунтификация, возможность изменить аватарку, данные профиля.
Админ пользователь, с возможностью выдавать роли пользователям.
Второй сервиса для манипуляции списками желаний, добавление новых, добавление предметов в список.
Третий сервис для организации мероприятий(самой игровой сессии). Приглашение пользователей, выбор желаемого списка желаний, жеребьевка.
